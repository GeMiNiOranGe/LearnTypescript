# LearnTypescript

> Caption: **[Node.js](https://nodejs.org/)** version must be newer than **20.0.0**

## How to build
- Step 1: intall library.
    ```
    npm install
    ```
- Step 2: run project.
    - Use `ts-node`
        ```
        npx ts-node .\index.ts
        ```
    - Use `tsc`
        ```
        npx tsc .\index.ts
        ```
        ```
        node .\index.js
        ```
